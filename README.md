## What is WALLET.md?

WALLET.md defines how an agent handles money — from traditional enterprise billing to DeFi/crypto autonomous transactions. As agents increasingly hire other agents, pay for API access, and manage budgets autonomously, WALLET.md provides the structured, auditable financial identity that makes this possible.

Without WALLET.md, an agent with financial authority is a liability. WALLET.md constrains spending, defines approval thresholds, and ensures financial actions are auditable. It's the difference between an agent with a corporate card and an agent with a blank check.

Create a WALLET.md for any agent that spends money, manages budgets, or participates in economic transactions with other agents or services.

## When to use WALLET.md

- Agents managing cloud infrastructure billing or API spend
- DeFi trading agents with on-chain wallets for autonomous transactions
- Agent-to-agent payment in marketplace environments (see HIREME.md)
